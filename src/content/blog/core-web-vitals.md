Le performance non sono solo un vezzo tecnico, ma un fattore cruciale per il SEO e la conversione degli utenti. Google utilizza i Core Web Vitals come segnale di ranking.

### I tre pilastri fondamentali:

- **LCP (Largest Contentful Paint):** Misura la velocità di caricamento percepita.
- **FID (First Input Delay):** Misura la reattività del sito.
- **CLS (Cumulative Layout Shift):** Misura la stabilità visiva.
